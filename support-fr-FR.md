# Assistance KanbanApp

Besoin d'aide avec KanbanApp ? Utilisez cette page pour obtenir de l'assistance, resoudre un probleme ou nous contacter.

## Contact

Pour les demandes d'assistance, questions ou signalements de bugs, ouvrez une issue :

https://github.com/atamayo/kanbanapp-legal/issues

Veuillez indiquer le modele de votre appareil, la version d'iOS, la version de l'app et une courte description du probleme.

## Questions frequentes

### Ou mes taches sont-elles stockees ?

KanbanApp stocke vos taches sur votre appareil. Si la synchronisation iCloud est activee, Apple peut synchroniser les donnees de l'app entre vos appareils connectes.

### Comment supprimer mes donnees ?

Vous pouvez supprimer les taches directement dans l'app. Vous pouvez aussi supprimer l'app de votre appareil afin d'effacer les donnees locales de l'app.

### Pourquoi l'app demande-t-elle l'acces a l'appareil photo, aux photos, au micro ou a la reconnaissance vocale ?

Ces autorisations sont utilisees uniquement lorsque vous choisissez des fonctions de capture rapide comme la photo, la numerisation, la voix ou la dictee.

### Comment signaler un bug ?

Ouvrez une issue de support avec le lien ci-dessus et decrivez ce qui s'est passe, ce que vous attendiez et les etapes pour reproduire le probleme.
